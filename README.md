# 07 — Risk Register

## Project Overview

Built a structured risk register for a fictional SMB (small-medium business) using a likelihood × impact scoring matrix aligned to NIST RMF principles.

## What I Did

- Identified 15 information security risks across people, process, and technology domains
- Scored each risk using a 5×5 likelihood × impact matrix (1–25 scale)
- Mapped existing controls to each risk and calculated residual risk scores
- Assigned treatment options: Accept, Mitigate, Transfer, or Avoid
- Produced a board-ready top 5 priority risk summary and control gap analysis

## Scoring Methodology

```
Risk Score = Likelihood (1–5) × Impact (1–5)

  20–25  →  Critical
  15–19  →  High
  8–14   →  Medium
  1–7    →  Low
```

## Key Skills Demonstrated

- Risk identification across people, process, and technology domains
- Quantitative risk scoring methodology
- Control mapping and gap analysis
- Residual risk calculation
- Executive-facing risk communication
