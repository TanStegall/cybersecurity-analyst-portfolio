## 01 — Threat Intelligence Report

**Folder:** [`01-threat-intelligence/`](./01-threat-intelligence/)

A professional threat intelligence report profiling a real-world threat actor group (APT29 / Cozy Bear). Includes TTPs mapped to MITRE ATT&CK, indicators of compromise, targeted industries, and analyst recommendations.

**Demonstrates:** Research methodology, MITRE ATT&CK navigation, structured analytical writing, IOC identification.

## 02 — SIEM Home Lab (Wazuh)

**Folder:** [`02-siem-lab/`](./02-siem-lab/)

Built a functional SIEM environment using Wazuh on a local virtual machine. Configured agents, ingested logs from a simulated endpoint, triggered alerts using known attack techniques, and documented tuning decisions.

**Demonstrates:** SIEM deployment and configuration, log source onboarding, alert triage, false positive reduction.

## 03 — Vulnerability Assessment

**Folder:** [`03-vulnerability-assessment/`](./03-vulnerability-assessment/)

Conducted a vulnerability scan of a home lab environment using Nessus Essentials. Findings were documented in a professional assessment report with CVSS scores, business context, and a prioritized remediation roadmap.

**Demonstrates:** Nessus operation, CVSS interpretation, risk-based prioritization, executive-facing report writing.

## 04 — Incident Response Playbook

**Folder:** [`04-ir-playbook/`](./04-ir-playbook/)

Authored a complete Incident Response playbook for a ransomware scenario following the NIST SP 800-61 framework. Covers detection, containment, eradication, recovery, and post-incident activity with decision trees for escalation.

**Demonstrates:** NIST IR framework, playbook authorship, escalation logic, professional documentation.

## 05 — Phishing Email Analysis

**Folder:** [`05-phishing-analysis/`](./05-phishing-analysis/)

Analyzed a simulated phishing email end-to-end: extracted and decoded headers, identified spoofed sender infrastructure, checked URLs and attachments against threat intel platforms, and produced a triage report.

**Demonstrates:** Email header analysis, IOC extraction, VirusTotal/urlscan.io usage, analyst triage reporting.

## 06 — Network Forensics (PCAP Analysis)

**Folder:** [`06-network-forensics/`](./06-network-forensics/)

Analyzed a suspicious PCAP file using Wireshark. Identified anomalous traffic patterns consistent with C2 beaconing, exfiltration indicators, and lateral movement. Produced a findings report with packet-level evidence.

**Demonstrates:** Wireshark proficiency, protocol analysis, anomaly identification, evidence documentation.

## 07 — Risk Register

**Folder:** [`07-risk-register/`](./07-risk-register/)

Built a risk register for a fictional SMB organization using a structured likelihood × impact scoring methodology. Includes threat identification, existing control mapping, residual risk scoring, and treatment recommendations.

**Demonstrates:** Risk management framework application, structured risk scoring, control gap analysis, Excel/reporting skills.
